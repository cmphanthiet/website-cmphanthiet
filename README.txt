# Website CM Phan Thiết

## Cấu trúc thư mục:
- index.html (trang chính)
- images/sample.jpg (ảnh mẫu)
- video/intro.mp4 (video nền mẫu)
- music/sample.mp3 (nhạc mẫu)

## Cách sử dụng:
1. Mở file index.html bằng trình duyệt để chạy website.
2. Thay thế:
   - Đổi hình trong thư mục images/
   - Đổi video trong thư mục video/
   - Đổi nhạc trong thư mục music/
3. Viết nội dung ở các phần <section> trong index.html.

Website chạy offline, không cần server.
